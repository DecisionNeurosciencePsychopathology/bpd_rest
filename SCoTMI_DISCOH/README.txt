-- In matlab, add the SCoTMI_DISCOH code to the path via the graphical menu, or via the command prompt as: 

addpath(‘/yourPath/SCoTMI_DISCOH’)

NB: Do not need to add any subfolders to path.

-- Then run the example, in the matlab command line prompt (>> ):
SCoTMI_DISCOH_example()

-- Copy the example file to your own working directory and edit it for different purposes.

-- Help is available for each file, just type e.g.:
help SCoTMI_DISCOH_calc
